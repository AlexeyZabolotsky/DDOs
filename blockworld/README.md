# Blockworld 3D

3D-симуляция блоков на Ursina: клетки, redstone, полевые оболочки, выделение групп полем и всплывающий редактор параметров блока.

Объединяет функционал пользовательской симуляции и HUD из пакета `redstone` (хотбар, модификаторы, raycast-взаимодействие).

## Установка

```bash
cd blockworld
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Управление

| Клавиша | Действие |
|---------|----------|
| `1`–`9` | Слоты хотбара (выделение, поле/группа, параметры, connect, conduct, …) |
| `ЛКМ` | Действие по текущему слоту / выделение |
| `ПКМ` | Поставить блок |
| `E` | Открыть popup параметров блока под прицелом |
| `F` | Выделить связанную группу блоков полем |
| `Tab` | Переключить форму поля (куб / сфера) |
| `Scroll` | Радиус поля (в режиме поля) |
| `Shift` + `1`–`9`, `-` | Инструмент redstone |
| `Shift` + `ПКМ` | Поставить redstone-компонент |
| `U` | Режим UI |
| `Delete` | Удалить выделенные блоки |
| `F5`–`F8` | Добавить полевую оболочку |
| `Shift` + `F9` | Убрать все оболочки |

## Выделение полем из группы

Режим **«Поле/группа»** (слот `2`):

1. Клик по блоку — flood-fill связанной компоненты (соседи + conduct/connect).
2. Пересечение с полем заданного радиуса вокруг блока.
3. Все попавшие блоки добавляются в `selected_blocks`, показывается полупрозрачная оболочка.

## Popup параметров блока

Клавиша `E` или слот **«Параметры»** открывает панель у курсора:

- Режим состояния: connect / conduct / generate / store
- Направление, значение данных
- Redstone тип и сила (0–15)
- Тип клетки, тип нейрона
- Флаг активности (0/1)

Кнопка **«Применить»** записывает параметры в блок без отдельного режима state input.

## Структура

```
blockworld/
  main.py
  blockworld/
    app.py              # Ursina-приложение
    params.py           # Применение параметров
    ui/
      extended_hud.py     # Хотбар (из redstone HUD)
      field_selector.py   # Выделение полем
      block_popup.py      # Popup параметров
    systems/
      block_manager.py
      cells.py
      redstone.py
      fields.py
      brain.py
```

## Связь с redstone

Пакет `../redstone` содержит отдельный 2D/3D redstone-движок. Blockworld использует тот же паттерн HUD и группового выделения, расширенный полевыми оболочками и popup-редактором.
